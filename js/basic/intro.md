# JavaScript 简介

JavaScript 是一种轻量级的、解释型的编程语言，主要用于 Web 开发。

## 历史

- 1995 年由 Brendan Eich 创建
- 最初命名为 Mocha，后改名为 LiveScript，最终定名为 JavaScript
- ECMAScript 是其标准化规范

## 运行 JavaScript

浏览器内嵌了 JavaScript 引擎，可以直接运行：

```html
<script>
    console.log("Hello, JavaScript!");
</script>
```

也可以使用 Node.js 在服务端运行：

```bash
node app.js
```
