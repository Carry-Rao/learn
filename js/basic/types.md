# 数据类型与运算符

JavaScript 有 7 种基本数据类型。

## 基本类型

```javascript
// 数字
let age = 25;
let price = 19.99;

// 字符串
let name = "Alice";
let greeting = `Hello, ${name}`;

// 布尔值
let isActive = true;

// null 和 undefined
let empty = null;
let notDefined;

// 对象
let person = { name: "Bob", age: 30 };
```

## 运算符

```javascript
// 算术运算符
console.log(10 + 5);   // 15
console.log(10 - 5);   // 5
console.log(10 * 5);   // 50
console.log(10 / 3);   // 3.333...

// 比较运算符
console.log(5 === "5"); // false（严格相等）
console.log(5 == "5");  // true（宽松相等）
```
