# 指针与引用

指针和引用让我们间接访问变量。

## 指针

指针保存变量的内存地址。

```cpp
int x = 42;
int* p = &x;   // p 指向 x 的地址

std::cout << p;     // 输出地址（如 0x7ffd...）
std::cout << *p;    // 解引用，输出 42

*p = 100;      // 通过指针修改 x
std::cout << x;     // 100
```

## 引用

引用是变量的别名，必须初始化且不能更改指向。

```cpp
int x = 42;
int& r = x;    // r 是 x 的引用

r = 100;       // 修改 x
std::cout << x;     // 100
```

## 指针 vs 引用

| 特性 | 指针 | 引用 |
|------|------|------|
| 是否可为空 | 可以（nullptr） | 不能 |
| 能否改变指向 | 能 | 不能 |
| 语法 | `*p` 解引用 | 直接使用 |
| 用途 | 动态内存、可选参数 | 函数参数避免拷贝 |

## 函数参数传递

```cpp
void swap(int* a, int* b) {   // 指针传参
    int tmp = *a;
    *a = *b;
    *b = tmp;
}

void swap2(int& a, int& b) {  // 引用传参
    int tmp = a;
    a = b;
    b = tmp;
}
```

建议：能用引用就用引用，需要可为空时用指针。
