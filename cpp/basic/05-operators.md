# 运算符

C++ 提供丰富的运算符用于操作数据。

## 算术运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加 | `a + b` |
| `-` | 减 | `a - b` |
| `*` | 乘 | `a * b` |
| `/` | 除 | `a / b`（整数相除得整数） |
| `%` | 取模 | `a % b`（取余数） |

```cpp
int a = 10, b = 3;
std::cout << a + b;   // 13
std::cout << a - b;   // 7
std::cout << a * b;   // 30
std::cout << a / b;   // 3（整数除法）
std::cout << a % b;   // 1
```

## 复合赋值运算符

```cpp
int x = 5;
x += 3;   // x = 8
x -= 2;   // x = 6
x *= 2;   // x = 12
x /= 3;   // x = 4
```

## 自增自减

```cpp
int i = 0;
i++;      // 后置自增：先用后加
++i;      // 前置自增：先加后用
```

## 比较运算符

```cpp
==, !=, <, >, <=, >=
```

返回 `bool` 值：`true` 或 `false`。
